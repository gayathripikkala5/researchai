# ResearchAI

A local full-stack research paper tracking and recommendation tool.

## Project Structure

```text
frontend/   React + Vite app
backend/    FastAPI app with SQLite storage
```

## Run Backend

```powershell
.\start-backend.bat
```

## Run Frontend

```powershell
.\start-frontend.bat
```

Open the frontend URL shown by Vite. The frontend expects the backend at `http://localhost:8000`.
